⚙️ One-time setup — add your Google OAuth Client ID

Open index.html and replace YOUR_GOOGLE_CLIENT_ID near the top of the <script> with your real Client ID. Here's how to get one:

  1. Go to console.cloud.google.com/apis/credentials
  2. Click Create Credentials → OAuth client ID → Web application
  3. Add your deployed URL (and http://localhost for local testing) to Authorised JavaScript origins
  4. Enable the Google Drive API in APIs & Services → Library

The app only ever reads and writes the single file it creates in your Drive (taskboard_data.json). It cannot access any other Drive content.
